DevsBurger API

E-commerce Fullstack Descrição: Sistema completo de delivery desenvolvido com Java e Spring Boot. A aplicação gerencia todo o ciclo do pedido: vitrine de produtos dinâmica, carrinho de compras com gerenciamento de sessão (HttpSession) e checkout. Utiliza Hibernate (JPA) para automação total do banco de dados MySQL e Thymeleaf para o frontend.
