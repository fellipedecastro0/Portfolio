# Portfólio de Engenharia de Software | Fellipe de Castro

Bem-vindo ao meu portfólio oficial! Este projeto foi desenvolvido para demonstrar minhas competências em desenvolvimento Fullstack, focando na transição de lógica de Backend para interfaces modernas de Frontend.

![Preview do Projeto](https://via.placeholder.com/800x400?text=Em+Breve+Print+do+Site)


##  Tecnologias Utilizadas

* **HTML5 Semântico**
* **CSS3 Moderno** (Glassmorphism, Variáveis CSS, Flexbox)
* **Bootstrap 5** (Layout e Componentes)
* **JavaScript** (Manipulação de DOM e Lógica de Tooltips)
* **Devicon** (Ícones da Stack)

##  Funcionalidades

* **Hero Section Interativa:** Apresentação com design moderno.
* **Tech Stack Hover:** Efeitos visuais (Neon/Glow) para destacar as tecnologias.
* **Glassmorphism:** Estilo de vidro fosco aplicado nos cards.
* **Responsividade:** Adaptado para mobile e desktop via Bootstrap.
